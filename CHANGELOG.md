# Changelog

## 1.5.2

- Removed bundled Google API key from the public plugin source.
- Switched the default Google translation path to a keyless endpoint.
- Updated author and repository metadata to isimsizman09.

## 1.5.1

- Added incoming auto-translate for new messages.
- Added per-channel incoming auto-translate override.
- Improved live preview with a calmer timed preview mode.
- Improved preview bar styling.
- Improved native select and option dark-theme compatibility.

## 1.5.0

- Added LRU-style translation cache behavior.
- Added duplicate request protection for message translation.
- Added stale response protection for live preview.
- Added quick target-language selector under translations.
- Added copy translation action.
- Added selected-text translation.
- Added recent visible messages translation.
- Added interface language setting.
- Added Turkish settings UI support.
- Added protected content settings for commands, code-only messages, and link-only messages.
- Improved original message restoration for replace and interactive modes.

## 1.4.0

- Added Google, Groq, and OpenRouter translation provider support.
- Added live preview.
- Added context menu translation.
- Added toolbar translation button.
- Added channel language overrides.
- Added theme-aware settings UI.
